# TEXT2PICBOT-Generating-Visuals-From-Words-with-GANs

For the Dataset please download https://www.vision.caltech.edu/datasets/cub_200_2011/

for Char-CNN-RNN please download https://drive.google.com/file/d/0B3y_msrWZaXLT1BZdVdycDY5TEE/view?resourcekey=0-sZrhftoEfdvHq6MweAeCjA

- Objective: To create TEXT2PICBOT, a project aimed at generating realistic images from textual descriptions using Generative Adversarial Networks (GANs).

- Project Development: Developed TEXT2PICBOT, a text-to-image synthesis system using GANs to transform detailed textual descriptions into lifelike bird images.

- Advanced Architecture Implementation: Implemented the StackGAN architecture, leveraging its two-stage design. This approach progressively enhances image resolution and realism, achieving a seamless translation of textual descriptions into visually convincing bird images.

- Impact: By employing the advanced capabilities of StackGAN, the project demonstrated significant improvements in the quality and accuracy of generated images, pushing the boundaries of what's possible in text-to-image synthesis and showcasing the potential of GANs in creating good visual content from text.
